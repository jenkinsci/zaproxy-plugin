Checkbox to signify if Reports will be generated (checked) or not generated (unchecked).


Generate Report - This generates a new report containing the alerts raised in a supported format (.xml, .html).
Note: This is the default method of report generation for ZAP.

Export Report - This generates a new customized report containing the alerts raised in a supported format (.xml, .xhtml, .json).
Note: This requires a plugin called Export Report which can be found in ZAP > Manage Add-ons > Marketplace.
